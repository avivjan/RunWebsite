Run with friends, anywhere

Run is a social running app for hosting and joining group runs with live location sharing, event coordination, and route guidance.

Create events, pick a track, mark runners as ready, and see everyone on the map in real time.

  • Host or join runs with ready-up and start controls
  • Guided routes or free runs with GPS tracking
  • Live positions of participants during events

Runs are summarized automatically with distance, duration, pace, speed, calories, and elevation insights. Friends and profiles make it easy to train together.

Highlights

  • Live map with participant locations
  • Create events and choose tracks
  • Automatic run summaries with elevation stats

Features

Run brings real-time group running, route guidance, event management, and social features together in one app.

Runner Experience

Live GPS Tracking

See your position and your friends on the map in real time during events.

Guided Tracks & Free Runs

Select a predefined track for guided runs, or head out for a free run with GPS logging.

Run Summary

Auto-calculated distance, duration, pace, speed, calories, and elevation stats with a shareable map.

Friends & Profiles

Search users, send friend requests, and view profiles to keep your running network connected.

Secure Authentication

Token-based sign-in keeps your account secure while enabling seamless API access from the app.

Event Hosting

Create & Manage Events

Pick a start location, choose a track, invite friends, and manage participants in one place.

Ready-Up & Start

Hosts can mark the event as ready, ensure runners are prepared, and start the run for synchronized tracking.

Real-time Positions

All runner positions are updated live during events, powered by reliable real-time messaging.

Social

Friends & Discovery

Search for users, manage friend requests, and plan group runs with your training partners.

User Profiles

View basic profile details and your connections to quickly navigate to friends and events.

Services

Run is powered by reliable cloud services for real-time events, secure APIs, and accurate mapping.

Azure SQL Database

Reliable and scalable storage for users, events, tracks, friendships, and activities.

SignalR

Real-time messaging to broadcast runner positions, event state changes, and reconnections during group runs.

Azure Functions

HTTP-triggered serverless APIs for authentication, events, tracks, and activity logging.

JWT Authentication

Secure token-based access from the mobile app to protect user data and sessions.

WebView Maps

High-performance map rendering inside the app for the live runner map and run summary route.

Maps & Geolocation

Accurate GPS and map layers to display tracks, locations, and participant markers.

Expo & React Native

Modern mobile framework for a smooth cross-platform running experience.

Resiliency

Timeouts, retries, and automatic reconnection keep runs stable even on flaky networks.

Scalable APIs

Efficient endpoints for events, tracks, users, friendships, and analytics.

Logic

Run uses Azure Functions for APIs, SignalR for real-time updates, Azure SQL for data, and a React Native client with WebViews for maps.

Source available in the project repository.
